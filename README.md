# BotLeague — India's Ultimate Robotics Arena

A pixel-faithful frontend clone of the BotLeague website built with:
- **React 18**
- **TypeScript**
- **Tailwind CSS v3**
- **Vite**

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Open in browser
# http://localhost:5173
```

## Build for Production

```bash
npm run build
npm run preview
```
